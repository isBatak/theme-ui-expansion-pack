# Stack

Stack

## Import

```js
import { Stack } from 'theme-ui-expansion-pack';
```

## Usage

Use the `spacing` prop to change the space between the elements.
You can use any `space` value from the theme.

```jsx live=true
<Stack spacing={4}>
  <Dummy sx={{ width: '100%', height: '40px' }} />
  <Dummy sx={{ width: '100%', height: '40px' }} />
  <Dummy sx={{ width: '100%', height: '40px' }} />
</Stack>
```

### Align to center

```jsx live=true
<Stack spacing={4} align="center">
  <Dummy sx={{ width: '40px', height: '40px' }} />
  <Dummy sx={{ width: '60px', height: '40px' }} />
  <Dummy sx={{ width: '80px', height: '40px' }} />
</Stack>
```

### Dividers

```jsx live=true
<Stack spacing={4} dividers>
  <Dummy sx={{ width: '100%', height: '40px' }} />
  <Dummy sx={{ width: '100%', height: '40px' }} />
  <Dummy sx={{ width: '100%', height: '40px' }} />
</Stack>
```

---

## Props

| Name       | Type                          | Default | Description |
| ---------- | ----------------------------- | ------- | ----------- |
| `as`       | "div" \| "ol" \| "ul"         | `div`   |
| `align`    | "left" \| "right" \| "center" |         |             |
| `dividers` | `boolean`                     | `false` |             |
