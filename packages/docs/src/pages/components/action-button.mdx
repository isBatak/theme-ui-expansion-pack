# Action Button

Enhanced button with options to show icons and loading state.

## Import

```js
import { ActionButton } from 'theme-ui-expansion-pack';
```

## Usage

```jsx live=true
<ActionButton>Button</ActionButton>
```

### Button Variant

Use the `variant` prop to change the visual style of the Button.

```jsx live=true
<ButtonGroup spacing={4}>
  <ActionButton variant="primary">Button</ActionButton>
  <ActionButton variant="secondary">Button</ActionButton>
</ButtonGroup>
```

### Button with Icon

You can add left and right icons to the Button components from popular libraries like
[react-icons](https://react-icons.netlify.com) and just pass it into the button.

```jsx live=true
<ButtonGroup spacing={4}>
  <ActionButton leftIcon={MdBuild}>Email</ActionButton>
  <ActionButton rightIcon={MdCall}>Call us</ActionButton>
</ButtonGroup>
```

### Button loading state

Pass `isLoading` prop to the Button component to show it's loading state.

```jsx live=true
<ButtonGroup spacing={4}>
  <ActionButton isLoading>Email</ActionButton>
  <ActionButton isLoading>Submit</ActionButton>
</ButtonGroup>
```
