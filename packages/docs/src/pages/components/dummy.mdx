# Dummy

Building block for creating skeleton layout.

## Import

```js
import { Dummy } from 'theme-ui-expansion-pack';
```

## Usage

```jsx live=true
<Card>
  <Text>
    <Dummy sx={{ width: '500px' }} />
    <Dummy sx={{ width: '300px' }} />
    <Dummy sx={{ width: '400px' }} />
  </Text>
</Card>
```
